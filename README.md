# Desktop Pet

一个可以远程陪伴、控制动作、发语音、开视频通话的桌宠 App。

## 它是做什么的

双方安装桌宠后，可以从桌宠托盘打开内置控制台互相连接；控制台也可独立部署到浏览器：

- 控制桌宠动作、表情和位置
- 发送文字或语音，让桌宠替你说话
- 发起语音 / 视频通话
- 查看对方电脑上的桌宠状态

## 适合的场景

- 异地时做一个常驻桌面的陪伴入口
- 给对方电脑放一个能互动的小角色
- 用网页远程控制桌宠，不需要反复让对方操作
- 后续扩展成更完整的虚拟陪伴、语音互动和自动更新应用

## 当前形态

这个项目目前是一个 v1 桌宠系统：

- `web/` 是同时内置进 Electron、也可独立部署的控制台
- `server/` 负责连接和转发
- `pet/` 是包含桌宠窗口和控制面板的 Electron 客户端
