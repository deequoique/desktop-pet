# 预录台词放这里

文件名按部位前缀分类，戳模型对应部位时会**随机播一句**：

```
voices/
  head_1.wav   ← 戳头时随机播
  head_2.wav
  body_1.wav   ← 戳身体时
  tail_1.wav   ← 戳尾巴时
  idle_1.wav   ← 保留给后续空闲语音
```

- 支持 `.wav` / `.mp3` / `.ogg` / `.m4a`
- 文件名前缀必须是 `head_` / `body_` / `tail_` / `idle_`（大小写不限，下划线必须）
- 推荐 30-50 句覆盖全部三个部位
- 录的时候保持自然语气、安静环境、相同距离（计划 §三 第 4 节）

加文件后重启 pet（或托盘 Reload）就会被扫到。
